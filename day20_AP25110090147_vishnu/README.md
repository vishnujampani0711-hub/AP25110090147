the program stores product ids, quantities and prices using 1d arrays.

a 3×3 rack matrix is used to analyse stock distribution.

products can be added only if the id is unique and values are positive.

searching checks the list and shows data if the id exists.

quantity updating allows adding or removing stock safely.

the rack report finds max, min and total items inside the 3×3 matrix.

stock value is calculated using quantity × price for each product.

the program finds the highest and lowest stock value among products.

all products can be displayed in a simple table format.

a menu loop lets the user pick operations until they choose exit.
