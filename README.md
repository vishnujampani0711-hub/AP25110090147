the code is a basic inventory system for a small shop.

it stores up to 10 products using arrays.

every product has an id, name, quantity, and price.

the program uses a menu so the user can choose actions.

the user can add new products to the inventory.

it can display all saved products with their details.

it calculates the total value of all items in stock.

it also finds which product has the highest and lowest value.

the user can search for a product by entering its id.

the loop keeps running until the user chooses the exit option.
